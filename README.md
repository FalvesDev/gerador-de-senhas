---

## Sobre

Ferramenta simples e funcional para gerar senhas fortes. Desenvolvida para praticar algoritmos de randomização e lógica condicional em JavaScript puro.

### O que aprendi

- Algoritmos de randomização com `Math.random()`
- Manipulação de strings e arrays em JavaScript
- Lógica de configuração via checkboxes
- Copiar para clipboard com a API `navigator.clipboard`

---

## Funcionalidades

- ✅ Comprimento configurável (até 30 caracteres)
- ✅ Letras maiúsculas e minúsculas
- ✅ Números
- ✅ Caracteres especiais
- ✅ Copiar para área de transferência com um clique
- ✅ Interface simples e responsiva
